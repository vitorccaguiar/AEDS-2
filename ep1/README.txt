*COMPILACAO*

- Para compilar o codigo eh necessario utilizar do comando: gcc EP1.c -o EP1

*USO*

- Para utilizar o programa que calcula a distancia de edicao eh necessario:

* Executar no terminal da seguinte maneira: ./EP1
* O programa ira pedir duas palavras utilizando scanf diferentes.
* Digite uma palavra, de ENTER, digite a segunda palavra, de ENTER novamente.
* O programa ira exibir a matriz preechida depois dos calculos e exibira um printf com a distancia de edicao.

- Observacoes de como esta implementado:

* A matriz eh formada de cima para baixo, o tamanho dos vetores serao impressos inicialmente na primeira linha e primeira coluna.
* A distancia de edicao sera o ultimo elemento da matriz.
